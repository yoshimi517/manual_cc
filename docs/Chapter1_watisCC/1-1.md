### Section.1-1 コンポーネントとしての位置づけ
ControlCenter は Communication Suite3 のシステムの構成・設定、マスタデータの登録・編集を行うための Web アプリケーションです。他に以下の役割を持っています。  

- 各コンポーネントの動作状態の把握や、動作をコントロールする制御サーバ
- ライセンス管理サーバ
- 他システムへの API の提供サーバ
